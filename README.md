# TF2-HA

Current TF2 Trading Hub build: 5.13.60 - flashy HA-connected cockpit UI.

Repository contains the TF2 Trading Hub Home Assistant add-on and optional SDA Bridge helper.

Safety defaults remain guarded/manual: no live trade accepts, no Steam confirmations, no unsafe automatic publishing.

Access modes: Home Assistant Ingress remains enabled, and the add-on also exposes an optional direct Web UI on port `8099` for debugging/full-page use.
